# Day 2 - Operators & Conditions

## 📌 Topics Covered:
- Arithmetic Operators (+, -, *, /, %)
- Comparison Operators (>, <, ==, ===)
- Logical Operators (&&, ||, !)
- if-else conditions

---

## 🧠 What I Learned:
- Operators help perform calculations and comparisons
- if-else is used to make decisions
- === is better than == (strict checking)

---

## ⚠️ Important Points:
- == checks only value
- === checks value + type (recommended)
- Always use proper conditions inside if()

---

## 💻 Examples:

### Even or Odd
```js
let num = 10;

if (num % 2 === 0) {
  console.log("Even");
} else {
  console.log("Odd");
}

# Day 3 - Loops

## 📌 Topics Covered:
- for loop
- while loop

## 🧠 What I Learned:
- Loop is used to repeat tasks
- for loop is best when we know count
- while loop is used for condition-based repetition

## 🔁 Loop Syntax:

for (let i = 1; i <= 10; i++) {
  console.log(i);
}

## ⚠️ Important Points:
- Always update variable (i++ / i--)
- Wrong condition can cause infinite loop

## 🧩 Key Concept:
Start → Condition → Run → Update
